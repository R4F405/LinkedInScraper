Scrapear un perfil (URL)

Pega el enlace de un perfil de LinkedIn para descargar los contactos de esa persona. Se abrirá Chrome en una ventana de Terminal.

Scrapear mis contactos

Inicia sesión con tu email y contraseña de LinkedIn. Se descargarán los contactos de tu perfil (quien inicia sesión). Se abrirá Chrome en una ventana de Terminal.

¿Cómo se usa?

  1. Scrapear un perfil (URL): pega el enlace de un perfil de LinkedIn y pulsa Iniciar para descargar los contactos de esa persona.
  2. Scrapear mis contactos: introduce tu email y contraseña de LinkedIn y pulsa Iniciar con mi sesión para descargar tus propios contactos.
  3. Se abrirá una ventana con Chrome; puede tardar varios minutos.
  4. Cuando termine, verás aquí abajo la lista de exportaciones. Pulsa Ver esta exportación para ver los datos y Descargar para guardar en Excel, PDF o CSV.
