¿Cómo se usa?

  1. Pega el enlace de LinkedIn en el cuadro de arriba.
  2. Pulsa Iniciar. Se abrirá una ventana con Chrome; puede tardar unos minutos.
  3. Cuando termine, verás aquí abajo la lista de exportaciones con los contactos.
  4. Pulsa Ver esta exportación para ver los datos y Descargar para guardar en Excel, PDF o CSV.
